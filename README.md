# 🎮 ib-lol

**턴제 1v1 — 안개 속의 라인전**

리그 오브 레전드의 라인전을 턴제로 재해석한 웹 게임.  
전장의 안개(Fog of War) 속에서 적의 위치를 예측하고, 스킬을 좌표에 조준하세요.

🔗 **플레이:** [ib-lol.vercel.app](https://ib-lol.vercel.app)

## 🎬 트레일러

https://github.com/user-attachments/assets/placeholder

## ✨ 특징

- **블라인드 동시턴제** — 양쪽이 서로의 행동을 모른 채 동시에 행동. 순수 예측 싸움
- **전장의 안개** — 적 위치 숨겨짐, 5턴마다 공개. [Letters from Whitechapel](https://boardgamegeek.com/boardgame/59959/letters-from-whitechapel)에서 영감
- **좌표 타겟팅** — 스킬을 유닛이 아닌 좌표에 조준 (빗나갈 수 있음!)
- **챔피언: 애니** — Q/W/E/R + 패시브(파이로매니아) 풀 스킬셋
- **레벨 시스템** — Lv.1~18, 스킬 포인트 수동 배분, 레벨별 스킬 스케일링
- **아이템/상점** — 12종 아이템, 소모품 포함, 6슬롯
- **미니언/몬스터** — 고정 미니언(1타 처치, 10턴 리스폰), 버프/에픽 몬스터
- **점멸(Flash)** — 턴 소모 없는 순간이동, 쿨타임 24턴
- **모바일 대응** — 반응형 그리드, 탭 프리뷰 시스템

## 🕹️ 조작

| 행동 | 설명 |
|------|------|
| 이동 | 인접 1칸 이동 |
| 평타 | 사거리 내 좌표 공격 |
| Q/W/E/R | 스킬 사용 (좌표 지정) |
| 점멸 | 3칸 이내 순간이동 (프리 액션) |
| 상점 | 골드로 아이템 구매 |

**모바일:** 첫 탭 = AoE 범위 프리뷰, 같은 셀 재탭 = 실행
